## Installation

```bash
npm install nova-react-sdk
# or
yarn add nova-react-sdk
```

Peer dependencies

- react >= 16.8
- react-dom >= 16.8

Environment variables (app build-time)

- `NOVA_ORG_ID`: your organisation UUID
- `NOVA_APP_ID`: your app UUID
- `NOVA_API_ENDPOINT`: Nova Manager API base (e.g., http://localhost:8000)

Example provider config is in [provider.md](./provider.md).
